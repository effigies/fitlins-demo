Support
-------

The primary documentation for this project is found at https://fitlins.rtfd.io/.

All bugs, concerns and enhancement requests for this software can be submitted here:
https://github.com/poldracklab/fitlins/issues.

If you have a problem or would like to ask a question about how to use FitLins,
please submit a question to `NeuroStars.org <http://neurostars.org/tags/fitlins>`_ with a
``fitlins`` tag.
NeuroStars is a platform similar to StackOverflow but dedicated to neuroscience and
neuroinformatics.

Development-related discussion primarily occurs on GitHub, but you may also email
the `neuroimaging@python.org <http://mail.python.org/mailman/listinfo/neuroimaging>`_
mailing list.
Please add ``[fitlins]`` to the subject line when posting on the mailing list.
