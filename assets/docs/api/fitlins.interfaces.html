Module contents¶

The core functionality of FitLins is organized into Nipype interfaces, which fall into three primary categories:

  • The bids interfaces are for interacting with BIDS datasets and models.

  • nistats interfaces make the Nistats estimators available to Nipype workflows.

  • visualizations interfaces wrap matplotlib, Seaborn and nilearn to produce figures for reports.

Additional utility interfaces are available for reshaping and collating the outputs of interfaces.
